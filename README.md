Forum-system-ApiAutoTest
 
论坛系统接口自动化测试项目
基于 Python 打造的接口自动化测试工程，覆盖论坛系统用户、文章、评论、消息等核心业务接口，实现接口功能验证、异常场景覆盖、响应结果断言等核心能力，适配面试展示与项目测试落地。
 
项目介绍
 
本项目针对 Forum-system 后端系统，采用Pytest测试框架结合Requests库构建接口自动化测试体系，遵循模块化、可复用的设计原则，支持用例批量执行、测试数据解耦、日志自动记录，完美复现测试工程师的工程化思维与技术能力。
 
核心技术栈
 
类别 技术/工具 核心用途 
测试框架 Pytest 用例组织、批量执行、夹具复用、断言校验 
接口请求 Requests 发送HTTP请求（GET/POST/PUT/DELETE）、处理响应 
数据管理 YAML 解耦测试用例与测试数据（账号、入参、预期结果） 
日志记录 logging 自动生成测试日志，定位接口请求与断言异常 
报告生成 Allure（可选） 生成可视化测试报告，展示用例通过率与执行详情 
版本控制 Git 代码管理与版本追溯 
 
 
核心功能覆盖
 
- ✅ 用户模块：注册（合法/非法账号）、登录（正确/错误密码）、个人信息查询与修改
- ✅ 文章模块：发布（必填项缺失/内容过长）、编辑、删除、分页查询、详情查询
- ✅ 评论模块：发表评论、回复评论、删除评论、文章评论列表查询
- ✅ 消息模块：未读消息查询、消息标为已读、全部消息查询
